# Project 1 – Insertion Sort

Verilen Dizi:  
[22, 27, 16, 2, 18, 6]

Aşamalar:  
1. [22, 27, 16, 2, 18, 6]  
2. [22, 27, 16, 2, 18, 6]  
3. [16, 22, 27, 2, 18, 6]  
4. [2, 16, 22, 27, 18, 6]  
5. [2, 16, 18, 22, 27, 6]  
6. [2, 6, 16, 18, 22, 27]

Big-O Gösterimi:  
Best Case: O(n)  
Average Case: O(n²)  
Worst Case: O(n²)

Time Complexity Yorumu (18 sayısı için):  
Sıralı dizide 18 ortalarda yer aldığı için average case kapsamındadır.
